Where should the proactive budget insight appear?

Same data, different placement — pick the one that feels most natural for the workflow

A

Banner at top of My Reports

A dismissible alert card loads above the expense list when budget usage crosses a threshold (e.g. ≥ 75%).

My Reports
⚠️ Engineering team Q2 travel budget: 87% used — ¥1,300 remaining of ¥10,000. View breakdown
Receipt · Marriott Shanghai · ¥1,280 · submitted
Receipt · Didi · ¥45 · approved

Pros

  • Highest visibility
  • No action required to see it

Cons

  • Dismissible = can be ignored
  • Not tied to a specific action
B

Proactive chat message from AI agent

When the user opens My Reports, the AI agent sends an unprompted message with the budget status — blending naturally into the chat they already use.

  • Feels like Ramp — agent initiates
  • Conversational, can ask follow-ups

Cons

  • Easy to miss if chat is collapsed
  • Slightly more complex trigger logic
C

Contextual warning during submission

When filling a new expense, the agent checks remaining budget for that cost center and shows a warning inline — before they submit.
